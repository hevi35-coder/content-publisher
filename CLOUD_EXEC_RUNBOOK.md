# Cloud Execution Runbook

클라우드 실행 문서는 아래 파일을 사용하세요.

- `/workspace/content-publisher/docs/CLOUD_EXEC_RUNBOOK.md`

Codex Cloud 에이전트 프롬프트 예시:

```text
/workspace/content-publisher/docs/CLOUD_EXEC_RUNBOOK.md를 기준으로 명령을 실제 실행해.
설명만 하지 말고 단계별 실행 결과를 요약해줘.
실패가 나면 diagnose 단계까지 수행해서 run_log.txt까지 생성해.
```
